Description of compatibility by stars

    To display the description of compatibility By the Stars, you must execute the command of the same name, which is located in the "â‰¡" menu.

    In this window, a description of the compatibility of two partners is formed by their birth stars: GUA, day of the month, date of the house. At the top of the window is the compatibility indicator, which gives a brief assessment of the compatibility for this method. The following colors are used to indicate the degree of compatibility:
    green - good compatibility;
    gray - medium compatibility;
    red - poor compatibility.
    The program calculates potential relationships between partners, which are inherent in them by nature (as Feng Shui says). However, in life, the nature of relationships can be influenced by other factors that are not considered in the program. For a more complete introduction, you can read the description. If desired, you can get acquainted with the characteristics of each partner.
    Specialists can do more detailed analysis using the Pillars and Star Compatibility windows.
Below is a more detailed description of star compatibility.

    Descriptions generated by the application can be saved to files using the penultimate button on the toolbar. The files obtained in this way can be sent to friends and acquaintances (see Demonstration of settings and work with the application "People's Feng Shui Compatibility" for Android).
    By clicking the last button on the toolbar, the description can be slightly corrected (if so desired). The description must be edited before saving it to a file.

    All descriptions are saved in the application folder and are available using the "Saved descriptions" command.
