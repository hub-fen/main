Feng Shui Forecasts

The structure of the main application window (Pillars)

    There is a toolbar at the top of the screen. The toolbar contains a menu that is displayed when you click on the "≡" button.
    The second button is used to save the description to the application folder. The last button is for editing the description. These buttons appear when viewing descriptions.
    The "◄" "►" buttons are used to change the calculated Chinese hour (when displaying the Pillars), to change the calculated day (when displaying the Forecast for day), to change the calculated month (when displaying the Forecast for month), to change the calculated year (when displaying the Forecast for year).
    The middle button with the name of the window is designed to quickly switch between windows (Pillars, Forecast for day, Forecast for month, Forecast for year).

    The main window of the application displays frequently used information that Feng Shui specialists need for personality analysis.
    It is vertically divided into 3 parts:
1) information about the person;
2) the pillars of the calculated date (Hour, Day, Month, Year, Luck);
3) pillars of the date of birth (Hour, Day, Month, Year).
    The absence of the pillar of the Hour of birth indicates that the hour of birth is not entered for this person (this information may not be known even to the person himself) or the place of birth (it is not entered in the absence of the hour of birth).
    Each element in the pillars of fate has a corresponding color.
    Element names (Fire, Earth, Metal, Water, Tree (Wood)) can be displayed in different ways:
- with a capital letter (Fire, Earth, Metal, Water, Tree (Wood)), if the element is yang;
- with a small letter (fire, earth, metal, water, tree (wood)), if the element is yin;
    Hidden heavenly stems are displayed directly below the earthly branches. Elemental names are displayed as the initial letter of its name. If the letter is large - the Yang element, the small letter - the Yin element. For example: t+w+E means yin tree, yin water and yang Earth).
    Above and below the pillars, the merging of the elements can be displayed (the name of the elements in capital letters: FIRE, EARTH, METAL, WATER, TREE (WOOD)) and collisions of the elements (confl).

Additional information displayed when clicking on some elements of the main window

    A red flickering rectangle around the personality element (the heavenly element of the day in the date of birth) warns of special combinations: KuiGang (FaiKong), Three mystics, Yang edge, Qi Transformation, Heaven and Hell Network - Intuition, Follow the leader, Superpowerful personality.
    To find out what this special combination is, you need to click on the heavenly element of the day, located in the red rectangle, and a description of the special combination will appear at the bottom of the window.

    If you click on any earth element in the pillars of the first or second partner, a description of the elements of the specified pillar will appear.
