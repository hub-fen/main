Forecast for the year

    In this window, a description of the estimated year for the analyzed person is formed, compiled according to various methods.
    Descriptions generated by the application can be saved to files using the second button on the toolbar. The files obtained in this way can be sent to friends and acquaintances (see Demonstration of settings and work with the "Feng Shui Calculator" application for Android).
    By clicking the third button on the toolbar, the description can be slightly corrected (if there is such a desire). The description must be edited before saving it to a file.

    All descriptions are saved in the "Documents" folder. Therefore, the application will need permission from the user that was discussed on the "Configuring at the first start" page.
