Feng Shui Forecasts

Personality characteristic

    To display the Personality characteristic window, you must execute the command of the same name, which is located in the "≡" menu.

   In this window, a characteristic of the analyzed person is formed by his date of birth, compiled using various methods.

    Descriptions generated by the application can be saved to files using the second button on the toolbar. The files obtained in this way can be sent to friends and acquaintances (see Demonstration of settings and work with the "Feng Shui Forecasts" application for Android).
    By clicking the last button on the toolbar, the description can be slightly corrected (if so desired). The description must be edited before saving it to a file.

    All descriptions are saved in the Documents folder. Therefore, the application will need permission from the user, which was discussed on the Configuring at the first start page.
