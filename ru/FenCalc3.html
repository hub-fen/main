Калькулятор Фэн-Шуй

Настройки приложения

    Для настройки отображения нужной информации и выполнения расчетов в соответствии с используемой методикой служит правое меню на панели инструментов "≡".
    Все команды, кроме "Настройки (Settings)", служат для быстрого переключения на нужный вид отображаемой информации (первые две группы команд) и желаемый вариант расчета (последние две группы команд) в приложении.

    Отображение стихий
    В данную группу входят следующие команды:
    Названия стихий - отображает названия элементов стихий с указанием принадлежности к инь (название с малой буквы) или ян (название с большой буквы) и названия животных в земных стихиях (название с малой буквы - инь, название с большой буквы - ян). Слившиеся стихии отображаются большими буквами;
    Аспекты стихий - отображает аспекты стихий (Личность, Интеллект, Богатство, Власть, Ресурсы) с указанием значения вычисленной благоприятности для каждой стихии;
    Иероглифы - названия стихий отображаются в виде иероглифов.

    Отображение скрытых небесных стволов
    Данная группа команд задает информацию, которая будет отображаться в нижнем элементе каждого столпа:
    Скрытые небесные стволы - небесные стволы, скрытые в земных ветвях. Названия стихий отображаются в виде начальной буквы из ее названия. Если буква большая - янская стихия, малая буква - иньская стихия. Например: д+в+З означает дерево инь, вода инь и Земля ян;
    Доминирующие скрытые небесные стволы - отображает элементы в скрытых небесных стволах, которые имеют наибольшее влияние;
    Фазы Ци 1-й степени - отношение дневной доминаты к земной ветви;
    Фазы Ци 2-й степени - отношение небесного ствола к земной ветви, на которой он «сидит»; 
    Фазы Ци 3-й степени - отношение доминирующего скрытого небесного ствола к земной ветви, в которой он скрыт;
    Номер в 60-летнем цикле - порядковый номер года в 60-летнем цикле китайского календаря.

    Слияния стихий 
    Данная группа команд управляет учетом слияния стихий. В нее входят три команды: 
    Без слияния - слияния стихий не будут учитываться в расчетах и соответственно стихии не будут изменяться, но слияния будут отображаться над и под столпами судьбы, расчетной даты и в части "Слияния и столкновения столпов Судьбы со столпами расчетной даты";
    В пределах даты - слияния будут учитываться в пределах даты рождения и расчетной даты. Слияния стихий между столпами двух дат учитываться не будут;
    Транзитные - слияния будут учитываться в пределах даты рождения, расчетной даты и между столпами двух дат.

    Продолжительность часа
Постоянная - каждый китайский час будет соответствовать двум астрономическим часам.
Переменная - длительность китайского часа будет зависеть от географической широты места, которое указано в качестве места рождения человека и места расчета (места пребывания человека).

Команда "Настройки (Settings)"

    Данная команда отображает диалог, содержащий три вкладки. На вкладке "ОБЩИЕ" можно настроить:

    Дворцы жизни - как будут располагаться Дворцы жизни:
    Традиционные - используется следующая последовательность Дворцов: Дом жизни, Братья сестры, Отношения, Дети внуки, Власть богатство, Место проживания, Служащие и зарплата, Слуги рабы, Болезни опасности, Этика религия, Движение, Родители;
    Новые - используется другая последовательность Дворцов: Дом жизни, Власть богатство, Братья сестры, Родители, Дети внуки, Слуги рабы, Отношения, Болезни опасности, Движение, Служащие и зарплата, Этика религия, Место проживания.
    Используется в окне "Следующие 12", в интерпретациях при нажатии на земную стихию столпа и при нажатии на название дворца.

    Расчет Удачи - вариант расчета столпа Удачи:
    Традиционный - классический расчет столпов Удачи согласно теории Фэн-Шуй;
    Новый - более точный расчет по методике Дегтярева В.И.

    Восход солнца - какой момент восхода солнца использовать в расчетах:
    Верхний край диска - когда над горизонтом появится верхний край солнечного диска;
    Центр диска - когда над горизонтом появится середина солнечного диска.

    Начало дня - с какого момента должен начинаться день:
    Начало Крысы - когда наступает китайский час Крысы;
    Середина Крысы - когда наступает вторая половина китайского часа Крысы.

    Названия фаз Ци - различные варианты названий фаз Ци.
    1-й вар: зародыш, вскармливание, рождение, купание в милостях, одевание пояса, поступление на службу, пик расцвета, увядание, болезнь, смерть, хранилище, обрыв;
    2-й вар: эмбрион, питание, рост и развитие, грязная вода, шапка и пояс, составление счастья, императорские свет., ослабление, болезнь, смерть, могила, исчезновение разлож.;
    3-й вар: нейтральные фазы Ци, нейтральные фазы Ци, светящиеся фазы Ци, умирающие фазы Ци, светящиеся фазы Ци, светящиеся фазы Ци, светящиеся фазы Ци, нейтральные фазы Ци, нейтральные фазы Ци, умирающие фазы Ци, умирающие фазы Ци, умирающие фазы Ци.

    Кнопка "Начальные настройки" предназначена для задания настроек по умолчанию.

    Кроме этого на этой вкладке можно переключить язык интерфейса (Русский, Английский) и указать место хранения пользовательской базы данных - локально в папке приложения или в общей папке Documents, доступной всем приложениям из серии Фэн-Шуй.
    При включении флажка "Общая база данных для всех приложений" если приложению не предоставлено разрешение для работы в общих папках устройства при первом запуске, система будет запрашивать у пользователя такое разрешение (см. "Настройка при первом запуске").
    При наличии локальной базы данных приложение будет выдавать запрос на копирование локальной базы данных в папку Documents.
    Локальная база данных хранится в папке приложения, не видна для других приложений Фэн-Шуй ("Прогнозы по Фэн-Шуй", "Совместимость по Фэн-Шуй") и удаляется при удалении приложения с устройства.
    Общая база данных хранится в папке "Documents", которая видна для других приложений Фэн-Шуй и не будет удаляться при удалении приложения с устройства. Это стандартная папка, предназначенная для хранения различных документов пользователя. В этой папке будут сохраняться описания, которые формирует приложение.

    На вкладке "РАСЧЕТЫ" можно настроить приложение на нужный вариант слияния стихий:
    Тройные - нужно ли учитывать слияние тройных стихий (отображаться будут, но в расчетах участвовать не будут);
    Приоритет тройных стихий - при использовании в расчетах слияния тройных стихий, какие слияния делать в первую очередь. Если флажок включен, сначала будет происходить слияние тройных стихий, затем двойные слияния, в противном случае будет наоборот;
    Учитывать изменение стихии личности - в результате слияния стихий может меняться стихия личности. Данный флажок определяет нужно ли учитывать в расчетах такую ситуацию.
    Кроме варианта слияния стихий, на этой вкладке можно указать:
    Учитывать пустые ветви - нужно ли в расчетах учитывать пустые земные ветви;
    Учитывать тип "Следую за лидером" - расчетное значение благоприятности элементов будет меняется, если определяется тип личности «Следуй за лидером»;
    Учитывать тип "Супер сильная личность" - нужно ли в расчетах учитывать соответствующий тип личности. При использовании этого типа личности необходимо указать и минимальную силу личности, выше которой будет учитываться этот тип.

    На вкладке "СТОЛПЫ" можно настроить степень влияния каждого столпа на результаты расчетов (от 0 до 1):
    Столп зачатия с коэффициентом: - нужно ли использовать в расчетах столп зачатия. Если этот столп будет использоваться, то с каким коэффициентом он будет участвовать.
    Столп жизни с коэффициентом: - нужно ли использовать в расчетах столп жизни. Если этот столп будет использоваться, то с каким коэффициентом он будет участвовать.
     Остальные столпы исключить из расчетов нельзя, можно только указать коэффициент, с которым этот столп будет участвовать в расчетах. При значении коэффициента равном 0 столп не будет влиять на результаты расчетов.
