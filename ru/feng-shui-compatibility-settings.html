Совместимость по Фэн-Шуй

Настройки приложения

    Для настройки приложения необходимо выполнить команду Настройки (Settings), расположенную в меню "≡".   
    Данная команда отображает диалог, в котором можно настроить:
- язык интерфейса (Русский, Английский);
- указать место хранения пользовательской базы данных - локально в папке приложения или в общей папке Documents, доступной всем приложениям из серии Фэн-Шуй.
    При включении флажка Общая база данных для всех приложений если приложению не предоставлено разрешение для работы в общих папках устройства при первом запуске, система будет запрашивать у пользователя такое разрешение (см. Настройка при первом запуске).
    При наличии локальной базы данных приложение будет выдавать запрос на копирование локальной базы данных в папку Documents.
    Локальная база данных хранится в папке приложения, не видна для других приложений Фэн-Шуй ("Прогнозы по Фэн-Шуй", "Калькулятор Фэн-Шуй") и удаляется при удалении приложения с устройства.
    Общая база данных хранится в папке Documents, которая видна для других приложений Фэн-Шуй и не будет удаляться при удалении приложения с устройства. Это стандартная папка, предназначенная для хранения различных документов пользователя. В этой папке будут сохраняться описания, которые формирует приложение.
