Прогнозы (Гороскопы) по Фэн-Шуй

Отправить базу данных


    База данных храниться в папке приложения и удаляется автоматически при удалении приложения с устройства.
    Для создания копии базы данных, а так же для передачи базы данных в другие приложения из серии Фэн-Шуй служит команда Отправить базу данных.
    При выполнении команды появится окно со списком приложений, которые могут принять для хранения (Диск), или отправки (Gmail) файлы базы данных.
    В списке приложений будут присутствовать приложения из серии Фэн-Шуй, если они установлены на устройстве (Калькулятор Фэн-Шуй, Совместимость по Фэн-Шуй), которым можно передать файлы базы данных. Передача базы данных в эти приложения приведет к замене базы данных, используемой этими приложениями.
    При использовании нескольких приложений из серии Фэн-Шуй рекомендуется базу данных изменять в одном из этих приложений (на Ваше усмотрение) и передавать в другие приложения с помощью этой команды.
    Данная возможность появилась после того, когда разработчики системы Android запретили приложениям получать доступ к общим папкам, если эта необходимость не относится к основной функции приложения.
