Прогнозы (Гороскопы) по Фэн-Шуй

Настройка при первом запуске

1-й шаг

    При первом запуске приложение запросит язык, который будет использоваться для отображения текста в различных окнах, включая названия в географической базе данных. После нажатия на кнопку Далее приложение перейдет ко второму вопросу, связанному с расположением пользовательской базы данных.

2-й шаг

    Приложение позволяет работать с локальной и общей пользовательской базой данных.
    Локальная база данных хранится в папке приложения, не видна для других приложений Фэн-Шуй ("Совместимость по Фэн-Шуй", "Калькулятор Фэн-Шуй") и удаляется при удалении приложения с устройства.
    Общая база данных хранится в папке Documents, которая видна для других приложений Фэн-Шуй и не будет удаляться при удалении приложения с устройства. Это стандартная папка, предназначенная для хранения различных документов пользователя. В этой папке будут сохраняться описания, которые формирует приложение.
    При использовании общей базы данных (необходимо включить флажок) потребуется приложению предоставить соответствующее разрешение.

Разрешение для работы с папкой Documents

    При включении флажка общей базы данных система Android запросит у пользователя разрешение на работу приложения "Прогнозы по Фэн-Шуй" с файлами, находящимися в общих папках, в том числе и в папке Documents (см. рисунок).
    Если принято решение использовать общую базу данных, то приложению обязательно нужно предоставить  разрешение, нажав на соответствующую кнопку РАЗРЕШИТЬ в запросе.
    При работе с приложением расположение базы данных можно будет менять, используя настройки приложения.

3-й шаг

    На этом шаге предлагается пользователю познакомится с возможностями приложения, посмотрев видео (кнопка Прогнозы (Гороскопы) по Фэн-Шуй) и закончить предварительные настройки.
