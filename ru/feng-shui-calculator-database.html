Калькулятор Фэн-Шуй

Работа с базой данных

    База данных предназначена для хранения информации о людях, анализ которых периодически необходимо делать. Для работы с базой данных предназначена команда База данных, которая находится в левом меню "≡" при выполнении которой отобразится окно диалога, который показан на рисунке.
    Первая строка - заголовок, содержащий название окна и две кнопки для закрытия диалога. Для выбора человека, по которому будет производиться расчет, нужно выделить его в списке и нажать на кнопку"v". Кнопка "х" просто закрывает диалог без каких либо изменений в приложении.
    Вторая строка управляет отображением людей в списке. Для этого она содержит выпадающий список с названиями имеющихся в базе групп людей. При выборе из этого списка нужной группы, нижний список будет содержать записи, относящиеся только к указанной группе. Для просмотра всех людей, имеющихся в базе данных из выпадающего списка нужно выбрать строку со словом Все.
    Если планируется в базе данных хранить информацию о многих людях, то лучше использовать их группировку. Для этого нужно нажать на зеленую кнопку ">", находящуюся во второй строке, и создать несколько групп для распределения людей. 
    Оставшуюся часть диалога занимает список людей. Справа внизу находится синяя кнопка "+", с помощью которой в базу данных добавляются новые люди. Остальные синие кнопки "v" и "-" появляются если нажать на интересующую строку в нижнем списке и немного подождать.
    Кнопка "-" предназначена для удаления выделенной строки из базы данных, а кнопка "v" для изменения информации о человеке.

    Если нажать на зеленую кнопку ">", находящуюся во второй строке диалога База данных отобразится диалог работы с группами, окно которого похоже на окно предыдущего диалога.
    В заголовке содержится только одна кнопка "x", которая закрывает диалог, а ниже идет список содержащихся в базе данных групп людей.
    Справа внизу находится синяя кнопка "+", с помощью которой в базу данных добавляются новые группы. Остальные синие кнопки "v" и "-" появляются если нажать на интересующую строку в списке и немного подождать.
    Кнопка "-" предназначена для удаления выделенной строки из базы данных, а кнопка "v" для изменения названия выделенной группы.

    Если нажать на синюю кнопку "+" или "v" в диалоге База данных то приложение предоставит возможность добавить в базу нового человека, или изменить информацию о нем, отобразив диалог Данные о человеке, который показан на рисунке слева. При добавлении нового человека диалог будет пустой.
    Назначение элементов в диалоге:
    поле Имя предназначено для ввода имени человека, которое должно быть уникальным во всей  базе данных; 
    поле Группы предназначено для отображения групп, в которые включен человек. При желании  человек может входить в несколько групп. Для включения человека в нужные группы необходимо нажать на кнопку ">" и отобразится немного измененный диалог Группы, показанный на рисунке ниже;
    флажок Время рождения предназначен для включения в расчет часового столпа и столпа Жизни в дате рождения человека, при этом необходимо задать и место рождения. Если место рождения не указано, часовой столп и столп Жизни вычисляться не будут. Время рождения задается путем нажатия на кнопку, справа в той же строке (диалог см. ниже);
    кнопка Дата рождения предназначена для задания даты рождения человека (диалог см. ниже);
    флажок Срок беременности включает вычисление столпа Зачатия. При включении данного столпа необходимо указать срок беременности матери в месяцах (обычно 9 месяцев);
    переключатель Пол предназначен для задания пола человека;
    кнопки Место рождения и Место расчета предназначены для ввода соответствующих данных о человеке (см. диалог Место рождения ниже).

    Если в диалоге Данные о человеке нажать кнопку ">", то отобразится немного измененный диалог Группы, который предназначен для включения человека в нужные группы (флажки в начале названия каждой группы). При изменении информации о группах необходимо будет нажать на кнопку "v", находящуюся в строке заголовка диалога (первая строка).
    Кнопки "+", "-", "v", находящиеся справа в списке групп выполняют те же функции, о которых речь шла выше.
    Для включения человека в нужные группы необходимо включить флажки вначале названия группы и нажать на кнопку "v".

        Данный диалог используется для ввода времени рождения. Время можно вводить непосредственно в верхнее поле или прокручивать колесики часов и минут, пока не получится нужное время.

    Данный диалог используется для ввода даты рождения. Дату можно вводить непосредственно в верхнее поле или прокручивать колесики дня, месяца и года, пока не получится нужная дата.

    Диалог для ввода информации о месте рождения и месте расчета. Название диалога меняется в зависимости от того, что вводится. Дальнейшее описание работы с этим диалогом будет идти применительно к месту рождения. Ввод информации о месте расчета аналогичен.
    Вводить информацию в этом диалоге нужно в порядке следования полей:
    выпадающий список Страна предназначен для ввода страны, в которой родился человек. Для этого можно ввести начало названия и нажать на кнопку "√" на клавиатуре устройства и приложение подберет ближайшее название из базы данных. Если при этом окажется не та страна, которая нужна, тогда необходимо нажать на кнопку "▼" из выпадающего списка выбрать нужную страну;
    выпадающий список Область предназначен для ввода региона, в котором родился человек. Действия в этом списке аналогичны предыдущему;
    выпадающий список Город предназначен для ввода населенного пункта, в котором родился человек. Действия в этом списке аналогичны предыдущему.
    После ввода населенного пункта, поля, которые расположены ниже, будут заполнены автоматически.
    При необходимости (в базе нет нужного населенного пункта) географические координаты места можно изменить. Часовой пояс при этом меняться не будет.
    На данный момент географическая база данных содержит информацию о 136600 населенных пунктах и таблицы изменения времени начиная с 1900 года.
