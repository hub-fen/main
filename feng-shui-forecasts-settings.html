Application settings

    To configure the application, you need to execute the Settings (Настройки) command located in the "≡" menu.
    This command displays a dialog in which you can configure:
- interface language (Russian, English);
- specify the storage location of the user database - locally in the application folder or in the Documents shared folder accessible to all applications from the Feng Shui series.
    When you checked the Shared database for all applications check box, if the application is not granted permission to work in the shared folders of the device at the first start, the system will ask the user for such permission (see Configuring at the first start).
    If there is a local database, the application will prompt you to copy the local database to the Documents folder.
    The local database is stored in the application folder, is not visible to other Feng Shui applications ("Feng Shui People Compatibility", "Feng Shui Calculator") and is deleted when the application is removed from the device.
    The shared database is stored in the Documents folder, which is visible to other Feng Shui applications and will not be deleted when the application is removed from the device. This is a standard folder for storing various user documents. This folder will store the descriptions that the application generates.
