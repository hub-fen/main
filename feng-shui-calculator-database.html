Working with the database

    The database is designed to store information about people, the analysis of which must be done periodically. To work with the database, the Data base command is intended, which is located in the left menu "≡", when executed, the dialog box shown in the figure will be displayed.
    The first line is a title containing the name of the window and two buttons to close the dialog. To select a person for whom the calculation will be made, you need to select him in the list and press the "v" button. The "x" button simply closes the dialog without any changes to the application.
    The second line controls the display of people in the list. To do this, it contains a drop-down list with the names of the groups of people available in the database. When you select the required group from this list, the lower list will contain records related only to the specified group. To view all the people available in the database, select the line with the word All from the drop-down list.
    If you plan to store information about many people in the database, then it is better to use their grouping. To do this, you need to click on the green ">" button located in the second line, and create several groups to distribute people.
    The rest of the dialogue is occupied by a list of people. At the bottom right, there is a blue "+" button, with which new people are added to the database. The rest of the blue buttons "v" and "-" appear if you click on the line of interest in the lower list and wait a little.
    The "-" button is used to delete the selected line from the database, and the "v" button is used to change information about a person.

    If you click on the green button ">" located in the second line of the Data base dialog, a dialog for working with groups will appear, the window of which is similar to the window of the previous dialog.
    The caption contains only one "x" button that closes the dialog, and below is a list of the groups of people in the database.
    At the bottom right is the blue "+" button, which adds new groups to the database. The rest of the blue buttons "v" and "-" appear if you click on the line of interest in the list and wait a little.
    The "-" button is used to delete the selected line from the database, and the "v" button is used to change the name of the selected group.

    If you click on the blue "+" or "v" button in the Data base dialog, the application will provide an opportunity to add a new person to the database, or change information about him, displaying the Person data dialog, which is shown in the figure on the left. When adding a new person, the dialog will be empty.
    Assigning elements in the dialog:
    Name field is intended for entering a person's name, which must be unique in the entire database;
    Groups field is intended to display the groups in which the person is included. If desired, a person can belong to several groups. To include a person in the required groups, click on the ">" button and a slightly modified Groups dialog will appear, shown in the figure below;
    checkbox Time of birth is intended to be included in the calculation of the hour pillar and the pillar of Life in the date of birth of a person, while the place of birth must also be specified. If the place of birth is not specified, the Pillar of Hour and the Pillar of Life will not be calculated. The time of birth is set by clicking on the button on the right in the same line (see the dialog below);
    button Date of birth is used to set the date of birth of a person (see the dialog below);
    checkbox Duration of gestation enables the calculation of the Conceive pillar. When this pillar is included, it is necessary to indicate the mother's gestational age in months (usually 9 months);
    Gender switch is designed to set the gender of a person;
    buttons Birth place and Place of calculation are intended for entering the relevant data about a person (see Place of Birth dialog below).

    If you press the ">" button in the Person data dialog, a slightly modified Groups dialog will be displayed, which is designed to include a person in the necessary groups (checkboxes at the beginning of the name of each group). When changing information about groups, you will need to click on the "v" button located in the title bar of the dialog (first line).
    The buttons "+", "-", "v" located on the right in the list of groups perform the same functions as described above.
    To include a person in the required groups, turn on the checkboxes at the beginning of the group name and press the "v" button.

    This dialog is used to enter the time of birth. The time can be entered directly into the top field, or you can scroll the hours and minutes wheels until the desired time is obtained. 

    This dialog is used to enter the date of birth. The date can be entered directly into the top field, or you can scroll the day, month, and year wheels until the desired date is obtained.

    Dialog for entering information about the place of birth and place of calculation. The dialog name changes depending on what is being entered. The further description of working with this dialog will be in relation to the place of birth. Entering the place of calculation is similar.
    You need to enter information in this dialog in the order of the fields:
    drop-down list Country is intended for entering the country in which the person was born. To do this, you can enter the beginning of the name and click on the "√" button on the keyboard of the device and the application will select the nearest name from the database. If the country is not the one you need, then you need to click on the "▼" button from the drop-down list to select the required country;
    drop-down list Region drop-down list is intended for entering the region in which the person was born. Actions in this list are similar to the previous one;
    drop-down list City is intended for entering the settlement in which the person was born. Actions in this list are similar to the previous one.
    After entering the settlement, the fields below will be filled in automatically.
    If necessary (there is no required settlement in the database), the geographical coordinates of the place can be changed. The time zone will not change.
    At the moment, the geographic database contains information about 136 600 settlements and tables of time changes since 1900.
