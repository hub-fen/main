Feng Shui People Compatibility

Partners characteristics

    In addition to the compatibility analysis, you can read the characteristic for each partner by executing the menu command with the name of the partner of interest, which is located in the Partners characteristics command group.

    Descriptions generated by the application can be saved to files using the penultimate button on the toolbar. The files obtained in this way can be sent to friends and acquaintances (see Demonstration of settings and work with the application "Feng Shui People Compatibility" for Android).
    By clicking the last button on the toolbar, the description can be slightly corrected (if so desired). The description must be edited before saving it to a file.

    All descriptions are saved in the Documents folder. Therefore, the application will need permission from the user, which was discussed on the Configuring at the first start page.
