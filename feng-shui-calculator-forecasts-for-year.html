Forecast for the year

    To display the Forecast for the year window, you must execute the command of the same name, which is located in the left menu "≡".

    In this window, a description of the estimated year for the analyzed person is formed, compiled according to various methods.
    Descriptions generated by the application can be saved to files using the second button on the toolbar. The files obtained in this way can be sent to friends and acquaintances (see Demonstration of settings and work with the "Feng Shui Calculator" application for Android).
    By clicking the third button on the toolbar, the description can be slightly corrected (if there is such a desire). The description must be edited before saving it to a file.

    All descriptions are saved in the application folder and are available using the "Saved forecasts" command.

In the year of the Ox, the "good old" methods and well-tested recipes work better than innovative ideas. The years of the Ox are also characterized by small conflicts based on mutual misunderstanding, which are usually resolved without much difficulty.
    In the year of the Ox, the Tiger experiences quarrels and confrontations. This is a big "show" of stubbornness and bad characters, leading to a large number of unresolved conflicts. Tigers must exercise patience if they do not want this year to become disastrous. It is not recommended to make changes to Feng Shui.
