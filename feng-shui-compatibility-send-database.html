Feng Shui People Compatibility

Send Data base

    The database is stored in the application folder and is deleted automatically when the application is removed from the device.
    To create a copy of the database, as well as to transfer the database to other applications from the Feng Shui series, use the Send Data base command.
    When the command is executed, a window will appear with a list of applications that can accept database files for storage (Disk) or sending (Gmail).
    The list of applications will contain applications from the Feng Shui series, if they are installed on the device (Feng Shui Forecasts, Feng Shui calculator), to which database files can be transferred. Passing the database to these applications will replace the database used by those applications.
    When using several applications from the Feng Shui series, it is recommended that the database be changed in one of these applications (at your discretion) and transferred to other applications using this command.
    This feature appeared after the developers of the Android system prevented applications from accessing shared folders, if this need is not related to the main function of the application.
