Feng Shui Forecasts

Configuring at the first start

1st step

    On first launch, the application will ask for the language that will be used to display text in various windows, including titles in the geographic database. After clicking the Next button, the application will move on to the second question related to the location of the user database.

2nd step

    The application allows you to work with local and shared user databases.
    The local database is stored in the application folder, is not visible to other Feng Shui applications ("Feng Shui People Compatibility", "Feng Shui calculator") and is deleted when the application is removed from the device.
    The shared database is stored in the Documents folder, which is visible to other Feng Shui applications and will not be deleted when the application is removed from the device. This is a standard folder for storing various user documents. This folder will store the descriptions that the application generates.
    When using a shared database (must be checked), the application will need to grant the appropriate permission.

Permission to work with the Documents folder

    When you checked the shared database checkbox, Android will ask the user for permission to use the "Feng Shui Forecasts" application with files located in shared folders, including the Documents folder (see figure).
    If it is decided to use a common database, then the application must be granted permission by clicking on the appropriate button ALLOW in the request.
    When working with the application, the location of the database can be changed using the application settings.

3rd step

     At this step, the user is invited to get acquainted with the capabilities of the application by watching the video (button Feng Shui Forecasts (Horoscopes)) and finish the preliminary settings.
