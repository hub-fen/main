Forecast for day

    To display the Forecast for day window, you must execute the command of the same name, which is located in the "â‰¡" menu, or press the necessary number of times on the middle button on the toolbar.

    In this window, a description of the settlement day for the analyzed person is formed. At the top of the window is the day indicator, according to the color of which (green - a good day, gray - medium, red - a bad day), you can quickly understand how favorable a given day will be for a person. The indicator contains a brief description of the day and recommendations.
    Below the indicator, in the form of diagrams, information is provided on favorable and unfavorable hours in a day and on favorable activities (what is better for a person to do on this day). Below is a description of the day, compiled according to various methods. Information on favorable hours and auspicious activities is also presented below in a different form.

    Descriptions generated by the application can be saved to files using the second button on the toolbar. The files obtained in this way can be sent to friends and acquaintances (see Demonstration of settings and work with the "Feng Shui Forecasts" application for Android).
    By clicking the last button on the toolbar, the description can be slightly corrected (if so desired). The description must be edited before saving it to a file.

    All descriptions are saved in the application folder and are available using the "Saved forecasts" command.
